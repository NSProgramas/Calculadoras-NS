Conversor de milhas terrestres

Como uma alternativa mais simples dos quilômetros e metros, a milha terrestre é bastante usada para calcular distâncias terrestres (como o próprio nome diz). Por serem as milhas mais usadas, são chamadas apenas de milhas e são tratadas como milhas "normais". A função dessa Calculadora Niduni é converter milhas terrestres para quilômetros e outros tipos de milhas.
